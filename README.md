# 🌍 Tourient — Tourist Safety App

Tourient is a smart tourist safety application designed to help travelers stay safe, connected, and informed during their journey.

The project focuses on emergency assistance, location-based safety support, and improving the overall travel safety experience.

---

# ✨ Features

* 🚨 Emergency SOS support
* 📍 Real-time location tracking
* 🔔 Safety alerts and notifications
* 🧭 Tourist assistance features
* 🔐 Secure authentication system
* ☁️ Scalable backend architecture

---

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/bappy-mallick/Tourient-tourist-safety-app.git
```

## Move to Project Directory

```bash
cd Tourient-tourist-safety-app
```

## Install Dependencies

```bash
npm install
```

---

# ▶️ Run the Project

## Start Backend

```bash
npm run server
```

## Start Frontend

```bash
npm start
```
